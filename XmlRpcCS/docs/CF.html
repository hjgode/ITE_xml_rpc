.NET Compact Framework Support

Microsoft .NET Compact Framework is a subset of the .NET Assemblies and friends geared towards running on PDA's and the like. Now, as near as I can tell, the ONLY way to get a hold of these assemblies for your x86 development host is to purchase Visual Studio .NET 2003.  I did find a distributable package for .NET CF on microsoft.com but it was only for the target platforms, none of which I own. So while anyone with net access can code C# for free, it seems you have to pony up for the .NET CF assemblies.  Go figure.

I did find, via Google, a document (NET_Compact_Framework.chm) that enumerated for each class in .NET CF what was different from the full .NET. 

Prerequisites to Build

  • Have .NET development tool and the .NET Compact Framework
  • have csc and friends in your PATH
  • Have nmake

To Build

Check the LDIR path in makefile.nmake. In a command shell: 
   nmake -f makefile.nmake
