Class Nwc.XmlRpc.XmlRpcBoxcarRequest

Class Index
Class that collects individual XmlRpcRequest objects and submits them as a boxcarred request. A boxcared request is when a number of request are collected before being sent via XML-RPC, and then are sent via a single HTTP connection. This results in a speed up from reduced connection time. The results are then retuned collectively as well.
See Also: Nwc.XmlRpc.XmlRpcRequest
