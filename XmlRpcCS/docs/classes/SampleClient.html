Class Sample.SampleClient

Class Index
A very basic XML-RPC client example. This is a basic client that calls several methods of the server including one intentionally bad one. One of the methods called works on the server.exe but is not exposed in the expose.exe, this gives you an idea of the error generated.

Public Methods

Main(System.String[])
Main application method. Simply sets up logging and then an XmlRpcRequest instance. Then Calls sample.Ping , sample.Echo and sample.Broken in sequence. The XmlRpcResponse from each call is then displayed. Faults are checked for.
WriteEntry(System.String,Nwc.XmlRpc.LogLevel)
LoggerDelegate compliant method that does logging to Console. This method filters out the LogLevel.Information chatter.
